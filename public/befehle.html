Neuen +Befehl anlegen
Befehle starten immer mit + — du gibst nur den Namen ohne + ein. Variablen wie [user] werden beim Ausführen ersetzt.
Zuschauer tippt dann: +hug
Kommagetrennt — mehrere Namen für denselben Befehl.
User bekommt diese F$ wenn er den Befehl nutzt. 0 = keine Belohnung.
Wie lange der User warten muss bis er den Befehl wieder nutzen kann.
Bei "Alle" darf jeder den Befehl nutzen.

Variablen-Verzeichnis
Diese Variablen kannst du im Antworttext verwenden — sie werden beim Ausführen automatisch ersetzt.
Variable Beschreibung Beispiel
BENUTZER
[user] Name des Users der den Befehl schreibt FaireWelt
[user.role] Rolle des Users (Admin / MOD / VIP / SUB / Chatter) MOD
[user.vip] Gibt "VIP" aus wenn der User VIP ist, sonst leer VIP
[user.followage] Seit wann der User dem Kanal folgt seit März 2022
[user.sub.since] Seit wann der User subscribed ist seit Januar 2023
[user.sub.months] Wie viele Monate der User schon subscribed ist 14
[user.bits.total] Gesamte Bits die der User je gespendet hat 2500
EMPFÄNGER
[empfaenger] Erster Parameter nach dem Befehl +hug @BottyFoxy → BottyFoxy

[empfaenger.fd] F$ Guthaben des Empfängers 420 F$
