Grundregeln
Der kleinste erlaubte Einsatz für ein Duell.
Der höchste erlaubte Einsatz für ein Duell.
Wie lange ein User nach einem Duell warten muss.
Nach dieser Zeit wird das Duell automatisch abgebrochen.
Admin Einstellungen
Gegner erhalten diesen Multiplikator, wenn sie den Admin schlagen.
Hinweis: Admin kann NICHT herausgefordert werden.
Verwaltung
Setzt alle Duell‑Daten zurück.
Beendet alle aktuell laufenden Duelle.
VIP Einstellungen
Erhöht die Gewinnchance eines VIP.
Normaluser erhalten diesen Multiplikator, wenn sie einen VIP schlagen.
MOD Einstellungen
Erhöht die Gewinnchance eines Moderators.
Normaluser erhalten diesen Multiplikator, wenn sie einen MOD schlagen.
SUB Einstellungen
SUBs erhalten diesen Multiplikator auf ihren Gewinn.
